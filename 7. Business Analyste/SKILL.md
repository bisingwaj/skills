# Business Analyste

## Aperçu
Cette compétence permet de produire des analyses business de qualité investisseur : business case, analyse de marché, projections financières, et positionnement concurrentiel.

## Quand utiliser cette compétence
- Rédiger un business case pour des investisseurs (Série A, B, ...).
- Analyser une opportunité de marché et calculer le TAM/SAM/SOM.
- Modéliser les projections financières sur 3 ans.
- Définir la stratégie go-to-market et les métriques clés.

## Structure d'un Business Case

### 1. Résumé Exécutif (2 pages max)
- Problème & Solution (3 points chacun)
- Marché : TAM/SAM/SOM
- Traction et métriques actuelles
- Équipe fondatrice
- Demande de financement

### 2. Analyse de Marché
```
TAM (Total Addressable Market) : Marché global accessible
SAM (Serviceable Addressable Market) : Part ciblée
SOM (Serviceable Obtainable Market) : Part réaliste
```

### 3. Modélisation Financière (3 ans)
| Métrique | An 1 | An 2 | An 3 |
|---|---|---|---|
| Revenu | - | - | - |
| Marge Brute | - | - | - |
| EBITDA | - | - | - |
| CAC | - | - | - |
| LTV | - | - | - |

### 4. Analyse Concurrentielle
- Matrice de comparaison (fonctionnalités, prix, positionnement)
- Différenciateurs clés et defensibilité
- Barrières à l'entrée

### 5. Projections et Usage des Fonds
- Montant demandé et structure (equity, SAFE, note convertible)
- Répartition par département (R&D, Sales/Marketing, Ops)
- Jalons à atteindre avec ce financement

## Bonnes pratiques
- **Quantifier tout** : Évitez les affirmations sans données.
- **Citer les sources** : Crédibilité des projections.
- **Scénarios** : Présenter les cas conservateur, base, optimiste.
- **Honnêteté** : Reconnaître les risques et les plans de mitigation.
