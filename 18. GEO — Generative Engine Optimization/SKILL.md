# GEO — Generative Engine Optimization

## Aperçu
Le **GEO** (Generative Engine Optimization) est l'art d'optimiser votre contenu pour être **cité dans les réponses des moteurs IA** (ChatGPT, Claude, Perplexity, Gemini). C'est la prochaine évolution du SEO.

## Quand utiliser cette compétence
- Optimiser du contenu pour apparaître dans les réponses des LLMs.
- Auditer la citabilité d'un article, d'un site ou d'une page.
- Définir une stratégie de contenu "AI-first".
- Construire l'autorité de marque dans les moteurs génératifs.

## SEO vs GEO

| Aspect | SEO | GEO |
|---|---|---|
| Objectif | Classement #1 | Être cité par l'IA |
| Plateforme | Google | ChatGPT, Claude, Perplexity, Gemini |
| Métriques | Rankings, CTR | Taux de citation |
| Focus | Mots-clés | Entités, données, clarté |

## Facteurs de Sélection des LLMs (RAG)

| Facteur | Poids estimé |
|---|---|
| Pertinence sémantique | ~40% |
| Correspondance de mots-clés | ~20% |
| Signaux d'autorité | ~15% |
| Fraîcheur du contenu | ~10% |
| Diversité des sources | ~15% |

## Contenu qui se Fait Citer

| Élément | Pourquoi ça marche |
|---|---|
| **Statistiques originales** | Données uniques et citables |
| **Citations d'experts** | Transfert d'autorité |
| **Définitions claires** | Facile à extraire |
| **Guides étape par étape** | Valeur actionnable |
| **Tableaux comparatifs** | Information structurée |
| **Sections FAQ** | Réponses directes aux questions |

## Checklist GEO

### Contenu
- [ ] Titres sous forme de question
- [ ] Résumé/TL;DR en haut de page
- [ ] Données originales avec sources
- [ ] Citations d'experts (nom, titre)
- [ ] Section FAQ (3-5 Q&R)
- [ ] Définitions claires et concises
- [ ] Horodatage "Dernière mise à jour"
- [ ] Auteur avec crédentiels

### Technique
- [ ] Schéma Article avec dates
- [ ] Schéma Person pour l'auteur
- [ ] Schéma FAQPage
- [ ] Chargement rapide (< 2.5s)
- [ ] Structure HTML propre et sémantique

## Construction d'Autorité (Entity Building)

| Action | But |
|---|---|
| Google Knowledge Panel | Reconnaissance d'entité |
| Mentions dans la presse | Signaux d'autorité |
| Cohérence des infos sur le web | Consolidation d'entité |
| Mentions dans l'industrie | Crédibilité sectorielle |

## Accès aux Crawlers IA

| Crawler | Moteur |
|---|---|
| GPTBot | ChatGPT / OpenAI |
| Claude-Web | Claude |
| PerplexityBot | Perplexity |
| Googlebot | Gemini (partagé) |

**Stratégie recommandée :** Autoriser tous les crawlers IA si vous souhaitez être cité.

## Mesure des Performances

| Métrique | Comment Suivre |
|---|---|
| Citations IA | Monitoring manuel dans les moteurs IA |
| Mentions "Selon [Marque]" | Recherche dans les LLMs |
| Trafic référé par l'IA | Paramètres UTM |
| Part de citation vs concurrents | Comparaison manuelle |

## Anti-Patterns à Éviter

| ❌ À ne pas faire | ✅ À faire |
|---|---|
| Publier sans date | Ajouter des horodatages |
| Attributions vagues | Nommer les sources précisément |
| Omettre les infos auteur | Afficher les crédentiels |
| Contenu superficiel | Couverture complète et profonde |

> **Règle d'or GEO :** L'IA cite le contenu qui est clair, faisant autorité et facile à extraire. Soyez la meilleure réponse disponible.
