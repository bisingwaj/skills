## 🚀 Installation

Chaque compétence est conçue pour être utilisée avec **[Antigravity](https://antigravity.google)**.

### Installation globale (recommandée)

Copiez le dossier de la compétence dans votre répertoire global Antigravity :

```bash
# Exemple : installer la compétence Brainstorming
cp -r "3. Brainstorming" ~/.gemini/antigravity/skills/brainstorming
```

### Installation locale (projet spécifique)

```bash
cp -r "3. Brainstorming" .agent/skills/brainstorming
```

## ✨ Format d'une Compétence

Chaque `SKILL.md` suit le format officiel Antigravity :

```yaml
---
name: nom-de-la-competence
description: "Description courte et précise de la compétence."
---

# Titre de la Compétence

## Quand utiliser cette compétence
...

## Instructions / Processus
...
```
